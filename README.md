--- System Requirements ---
- No specific system requirements. This should work on any modern system. Installation only tested on Windows 11.

--- Installation Steps  ---
- Install Python 3.12.10
- Install pip
- Install project requirements by running "pip install -r requirements.txt" in the project directory

--- Usage Instructions  ---
- the emails directory contains sample .eml files. You can add your own there.
- Run the project by running "python main.py" in the project directory
- A "reports" directory will be generated, which contains the reports for each .eml found in the emails directory.
